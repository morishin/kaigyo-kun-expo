# 改行くん Expo ver.

https://play.google.com/store/apps/details?id=me.morishin.kaigyokun.android

## Run

1. Launch packager
    ```sh
    exp start
    ```

1. Launch Android emulator by Genymotion

1. Launch app with Expo app in emulator
    ```sh
    exp android
    ```
## Generate apk

```sh
exp build:android
```
